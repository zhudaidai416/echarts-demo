# echarts-demo

## 介绍

ECharts 数据可视化项目 - 大屏数据可视化展示

## 技术

- 基于 flexible.js + rem 智能大屏适配
- VScode cssrem插件
- Flex 布局
- Less 使用
- 基于ECharts 数据可视化展示
- ECharts 柱状图数据设置
- ECharts 地图引入
